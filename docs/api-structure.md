# API Structure

## Health
- `GET /api/health`

## Dashboard
- `GET /api/dashboard/summary`

## Customers
- `GET /api/customers`
- `POST /api/customers`
- `PATCH /api/customers/:id`
- `DELETE /api/customers/:id`

## Payments
- `GET /api/payments`
- `POST /api/payments`
- `GET /api/payments/defaulters`
- `POST /api/payments/monthly-reset`

## Attendance
- `GET /api/attendance`
- `POST /api/attendance`

## Walk-ins
- `GET /api/walk-ins`
- `POST /api/walk-ins`

## AI
- `POST /api/ai/query`
